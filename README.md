This project predicts whether a volunteer is highly engaged based on volunteer activity data using Machine Learning.

* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

## Features

* Volunteer Data Analysis

* Engagement Prediction
* Machine Learning Classification
